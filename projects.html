Gemini (Chrome API)

This is to showcase the Gemini Nano browser API for Chrome (⚠️ Currently only works on Chrome canary)

How to enable:

  • chrome://flags/#optimization-guide-on-device-model -> Change to Enabled BypassPerfRequirement
  • chrome://flags/#prompt-api-for-gemini-nano -> Change to Enabled
  • Re-launch Chrome canary
  • If this doesn't workthen try the following:
    • chrome://components -> ctrl + f
      "On Device Model"
    • Click the "Update" button and wait till it's done.
